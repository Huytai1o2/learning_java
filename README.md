# learning_java

A personal repository for learning the Java programming language — notes, exercises, and small practice projects.

## Overview

This repo is a sandbox for working through Java fundamentals and building up to more
advanced topics. Code is organized by topic so each concept can be explored in isolation.

## Getting Started

### Prerequisites

- [JDK 17+](https://adoptium.net/) (any recent LTS works)
- An IDE such as [IntelliJ IDEA](https://www.jetbrains.com/idea/) (this project includes `.idea/` config)

### Clone

```bash
git clone https://github.com/Huytai1o2/learning_java.git
cd learning_java
```

### Compile & Run

For a standalone class:

```bash
javac -d out path/to/MyClass.java
java -cp out MyClass
```

## Topics

As the repo grows, examples will cover:

- Basics — variables, types, operators, control flow
- Methods & classes — OOP, inheritance, interfaces
- Collections — `List`, `Map`, `Set`, generics
- Exceptions & I/O
- Streams & lambdas
- Concurrency
